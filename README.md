# 🇵🇭 IOAI Philippines 2026

**Official repository for the IOAI Philippines 2026 national selection pipeline** -- problems, datasets, and evaluation notebooks used to select the Philippine delegation to the [International Olympiad in Artificial Intelligence (IOAI)](https://ioai-official.org/).

---

## 📂 Repository Structure

```
2026/
├── Selection/       # Team Selection round
├── Semi-Finals/     # Semi-finals round problems and evaluation
└── Finals/          # Finals round problems and evaluation
```

Each folder contains task descriptions, starter code, datasets, and grading rubrics used at each stage of the national selection.

---

## 🏆 About IOAI Philippines

IOAI Philippines is the national program that identifies and trains Filipino students for the International Olympiad in Artificial Intelligence. The program is run under **Ateneo BUILD** (Business Insights Laboratory for Development) at the John Gokongwei School of Management, Ateneo de Manila University.

Each year, students compete across multiple rounds -- from an initial eliminations to semi-finals and finals to selection -- to earn a spot on the Philippine national team.

---

## 📅 2026 Competition Rounds

| Round       | Folder         | Description                                            |
| ----------- | -------------- | ------------------------------------------------------ |
| Semi-Finals | `Semi-Finals/` | Shortlisted students compete in more advanced AI tasks |
| Finals      | `Finals/`      | Top finalists compete for national team placement      |
| Selection   | `Selection/`   | Team Selection Round                                   |
